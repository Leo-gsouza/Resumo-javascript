Variavel é um espaço reservado para receber um valor

String

Existem 3 formas de delimitar uma string » " ' ´

Nome da variavel

As variaveis são chamadas de indentificadores e possuem regras para nomea-las

  • Podem começar com uma letra, cifrão$ e subblinhado_
  • Não podem começar com numeros
  • É possivel usar letras e numeros
  • É possivel usar acentos e simbolos
  • Não pode ter espaço
  • Não podem ser palavras reservadas

Dicas para criar nome de variaveis

  • Maiuscula e miniscula fazem diferença
  • Usar nomes coerentes

Será usado o nodejs para trabalhar com as variaveis. Ele pode ser acessado pelo terminal do VSCODE, basta digitar node no terminal

Varialvel guarda todo tipo de dado que podem ser STRINGS, NUMBER(infinity/NaN) ou BOOLEAN(null/undefined) Object , Function

Usando o typeof você sabe o tipo da variavel

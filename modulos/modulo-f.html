Variaveis compostas

Variaveis compostas devem ser capazes de armazenar varios valores em uma mesma estrutura

pelo que entendi variaveis compostas são listas. Exemplo: automoveis = [palio, gol, uno, corsa, etios]. Observação: A lista sempre se inicia no indice 0. palio[0] gol[1] e assim por diante

Acrescentar um novo elemento na lista usando indice » automoveis[5] = corsa.

A lista vai ficar assim: automoveis[palio, gol, uno, corsa, etios, corsa]

Acrescentar um novo elemento no fim da lista(away) » automoveis.push('Up').Neste caso o Up será adicionado no fim da lista

Saber o comprimento da lista(away) » automoveis.length .observação: Note o que o length não tem parenteses, pois ele não é um metodo e sim um atributo

Para organizar os elementos dentro da lista em ordem cresente » automoveis.sort()

duas maneiras de mostrar cada elemento dentro de uma lista(away)

1ª Usando for.

                
for(contador = 0; contador < qtdd; contador++){
    console.log(`${contador+1}º » ${automoveis[contador]}`)}

2ª Usando o for in. observação: muito mais pratica

                
for(contador in automoveis){
    console.log(`${contador}º » ${automoveis[contador]}`)}

Encontrar posição de um elemento dentro da lista » automoveis.indexOf('uno'). Vai mostrar a posição(chave) que se encontra o elemento 'uno'. Caso não seja encontrado o valor dentro da lista ele vai retornar -1

Aula 16

Funções

funções são coisas que fazemos no dia a dia. Exemplo: Sua mãe te da a função de ir comprar leite no mercado. Para concluir essa função você vai prcisar:
1 colocar uma roupa adequada.
2 ir até o mercado.
3 procurar o leite.
4 pagar pelo leite.
5 voltar para casa

Toda função tem uma chamada, que no caso acima foi a chamada da mãe dizendo: comprar(dinheiro, leite). Dentro da função tem dois parametros que são: Leite e o dinheiro para compra-lo

Para conseguir realizar a função tem a ações, que no caso seria o passo a passo para comprar o leite

Temos o retorno que no caso seria o resultado desejado da função, ou seja, a mãe recebendo o leite

                
função comprar(parametro1, parametro2){
    pegar o dinheiro
    colocar uma roupa
    ir ate o mercado
    procurar o leite
    encontrou o parametro2? true
    tem dinheiro para o parametro2? true
    pagar pelo parametro2 com o parametro1
    retonar para casa
    entregar o parametro2 para mãe

    retorno = parametro1 comprado e entregue

    Agora a mãe do garoto vai fazer a chamada da função
    comprar(dinheiro, leite)
}

No exemplo acima a cghama inseriu no parametro 1 = dinheiro e no parametro 2 = leite. O retorno será Leite comprado e entregue

Variavel recebendo função

Uma variavel pode receber uma função

                
let variavel = function(parametro1, parametro2){
    return parametro1 + parametro2
}
console.log(variavel(8,11))

observação: A variavel carrega a função como seu valor. Sempre que chamar variavel é necessário abrir o parentes e inserir os parametros caso a função carregue eles.

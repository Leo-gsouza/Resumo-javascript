Trabalhando com somas.

Criado duas variaveis usando o comando window.prompt, a variavel n1 e n2. O objetivo é ter a soma das duas variaveis

var n1 = window.prompt('Digite o 1º número: ')
var n2 = window.prompt('Digite o 2º número: ')

observação: O sinal de + é usado tanto para concatenação como para adição. O JS vai gerar o resultado de acordo com o contexto em que o sinal vai estar inserido, exemplo: Se o sinal estiver entre dois números number + number, ele vai somar os valores . Se o sinal estiver entre uma string e um número string + number, ele vai concatenar.

Outa observação é que o comando window.prompt retorna uma string e não um número, então sempre que for usado o comando com objetivo de ter um valor numero será necessário converte-lo para number usando:

Number.parseInt(número)
Number.parseFloat(número)

Versão correta para trabalhar com números usando o comando prompt

var n1 = Number. parseInt(window. prompt('Digite o 1º número: '))
var n2 = Number. parseInt(window. prompt('Digite o 2º número: '))

observação: No JS recente pode usar apenas o Number sem a necessidade do Int ou Float, a não ser que queira forçar uma conversão para o tipo desejado

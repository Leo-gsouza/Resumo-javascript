Aprendendo a usar DOM em JavaScript

Clique em mim

Clique em mim foi criado com javascript usando o seguinte comando:

            
script
    var indentificador = window.document.getElementById('clicar')
    indentificador.style.background = 'lightblue'
    indentificador.style.color = 'black'
script

A ultima linha (está escrito assim) foi criada por javascript com o seguinte comando:

            
script
    var p1 = window.document.getElementsByTagName('p')[0]
    window.document.write(`Está escrito assim: ${p1.innerText}`)
script

Foi utilizado o getElementsByTagName e entre parenteses o p de paragrafo seguido de um 0 entre cochetes que referencia o 1º paragrafo do arquivo.

Este código criado com o metodo MARCA pegou o conteudo do 1º paragrafo p[0] e replicou ele usando write innerText
